# 📊 Blinkit Dashboard Project

## 🚀 Overview
Welcome to the Blinkit Dashboard—a data visualization and analytics tool built using Microsoft Excel. This dashboard offers key insights into sales performance, product distribution, outlet segmentation, and customer ratings for Blinkit, India's last-minute grocery delivery platform.

## 📁 Project Features
- 📌 Filterable dashboard based on:
  - **Outlet Size**: High, Medium, Small
  - **Outlet Location**: Tier 1, Tier 2, Tier 3
  - **Item Type**: Fruits & Vegetables, Dairy, Breads, Canned, etc.

- 📈 Key metrics visualized:
  - Total Sales: `$1.20M`
  - Average Sales: `$141`
  - Number of Items: `8523`
  - Average Rating: `4.0/5`

## 📊 Visualizations Included
| Visualization Type         | Description |
|----------------------------|-------------|
| Line Chart                 | Sales over years from 2011 to 2022 |
| Pie Charts                 | Fat Content breakdown; Outlet Size distribution |
| Bar Charts                 | Item Type sales; Fat Content by Outlet Tier; Outlet Location sales |
| Comparative Charts         | Outlet Types and their sales & items distribution |

## 🧮 Insights
- **Top-Selling Item Types**: Fruits & Vegetables, Snack Foods, Household
- **Outlet Performance**: Highest revenue from Small Outlet Size & Tier 1 Location
- **Fat Content Preferences**: Regular fat items account for 65% of total sales

## 💾 File Details
- Format: `.xlsx`
- Compatibility: Microsoft Excel 2016 or later
- File includes slicers and dynamic charts for interactive filtering

## 🛠 How to Use
1. Open the Excel file
2. Use filter panels to customize views
3. Explore each visualization to analyze Blinkit's sales and inventory data
4. Use slicers to toggle across different Outlet Sizes, Locations, and Item Types
